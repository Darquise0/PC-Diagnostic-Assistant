# PC Diagnostic Dashboard

A modern Windows desktop application built with C# and .NET that brings together essential PC diagnostic information into one intuitive interface.

## About

Diagnosing Windows PCs often requires jumping between multiple built-in tools such as Task Manager, Device Manager, Event Viewer, Resource Monitor, Settings, and Command Prompt. This project aims to simplify that experience by providing a single dashboard that presents important system information while helping users understand what it means.

Rather than simply displaying technical data, the long-term goal is to make PC diagnostics more accessible by providing clear explanations, identifying potential issues, and offering practical troubleshooting guidance.

## Project Status

🚧 This project is currently in active development.

The repository is in the planning stage while the architecture, project structure, and initial feature set are being designed before implementation begins.

## Planned Features

* System overview
* CPU information and utilisation
* Memory usage
* Storage information
* Running processes
* Network diagnostics
* Windows information
* Exportable diagnostic reports
* User-friendly explanations of detected issues
* Suggested troubleshooting steps
